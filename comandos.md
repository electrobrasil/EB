Comandos do EBbot:
=========

X especifique um numero,
algo entre ( ) são opicionais e
@ para menção

Hosts
-------

|Comando | Argumento |  Descrição |
|:------:|:---------:|:--------------------------------------:|
|!roulettepos | X | define a posição que a roleta moverá o ganhador |
|!cleardb | | limpa o banco de dados local do bot e o restaura.|
